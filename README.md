# 🎮 Простая текстовая приключенческая игра

текстовая игра на джава скрипт, написанная на JavaScript. Игрок принимает решения с помощью ввода текста или/и цифр и пытается дойти до успешного финала.

### Клонирование репозитория
Скопируйте проект к себе на компьютер:
```
git clone мой репозиторий
```
### перед запуском 
Скопируйте проект к себе на компьютер:
```
npm install -g http-server
npm install express   
```
### Запуск
Перейдите в папку с проектом и запустите главный файл:
```
запустите html
```

## 🎮 Как играть

1. Читайте текст истории на экране.
2. Вводите цифру или слово для выбора действия.
3. Нажимайте `Enter`, чтобы подтвердить ход.
4. Постарайтесь выжить и победить!
